# 单项目推荐封面布局模板

## 布局规格
- 尺寸：1080 × 1080 (1:1 方形) 或 1920 × 1080 (16:9 横向)
- 分辨率：72 DPI (在线) / 300 DPI (印刷)

## 配色方案

### 方案1：科技蓝（适合开发工具）
- 背景：#F5F7FA（浅灰）
- 主色：#007AFF（亮蓝）
- 强调色：#5856D6（紫蓝）
- 文字：#1D1D1F（深黑）
- 次级文字：#8E8E93（灰色）

### 方案2：活力橙（适合效率工具）
- 背景：#FFF8F0（浅橙）
- 主色：#FF9500（橙色）
- 强调色：#FF3B30（红色）
- 文字：#1D1D1F（深黑）
- 次级文字：#636366（深灰）

### 方案3：极简黑白（适合严肃工具）
- 背景：#FFFFFF（纯白）
- 主色：#000000（纯黑）
- 强调色：#007AFF（蓝色点缀）
- 文字：#1D1D1F（深黑）
- 次级文字：#8E8E93（灰色）

## 字体规范

### 标题
- 字号：48px - 72px
- 字重：Bold（700）
- 字体：PingFang SC, Helvetica Neue, Arial

### 副标题
- 字号：32px - 48px
- 字重：Medium（500）
- 字体：PingFang SC, Helvetica Neue, Arial

### 列表项
- 字号：24px - 32px
- 字重：Regular（400）
- 字体：PingFang SC, Helvetica Neue, Arial

## 设计元素

### 装饰元素
- 代码块：等宽字体，浅色背景
- 箭头图标：指向右侧，引导视线
- 星星图标：⭐ 用于Stars数量
- 闪光图标：✨ 用于功能点

### 对齐规范
- 主标题：居中对齐
- 副标题：居中对齐
- 列表项：左对齐
- 图片：居中对齐或左对齐
